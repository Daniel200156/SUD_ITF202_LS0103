Klassifizierung von Endgeräten

Netzwerkinfrastruktur ist dreiteilig: 

- Geräte (Hardware)
	-> Endgerät ist entweder Quelle oder Ziel einer Nachricht über das Netzwerk
		-> Indentifiziert durch Adresse
- Medien (Hardware verbindungen)
	-> Physische Komponente

- Dienste (Software)
	-> Alltägliche Netzwerkanwendungen, Betrieb eines Nezwerks

- Edit von dem File um einen neuen Feature Branch zu erstellen
- Weiteres Edit um den Feature Branch weiterzuführen
- Weiteres Edit, diesmal von Zuhause für die Hausaufgabe
